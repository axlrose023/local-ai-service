ПОЛІТИКА ВИКОРИСТАННЯ ЛОКАЛЬНОЇ МЕРЕЖІ ТА ІНТЕРНЕТУ

1. ПІДКЛЮЧЕННЯ ДО WI-FI
1.1. Для співробітників доступна мережа "UDO_Corporate".
1.2. Пароль для доступу: SuperSecure2024! (змінюється раз на 3 місяці).
1.3. Гостьова мережа "UDO_Guest" (без пароля) призначена лише для клієнтів. Підключення корпоративних ноутбуків до гостьової мережі ЗАБОРОНЕНО.

2. VPN ТА ВІДДАЛЕНИЙ ДОСТУП
2.1. Адреса VPN-шлюзу: vpn.udo.local
2.2. Для підключення використовується Cisco AnyConnect.
2.3. При проблемах з підключенням (помилка 403) необхідно скинути кеш DNS командою "ipconfig /flushdns".

3. ОБМЕЖЕННЯ
3.1. Заблоковано доступ до соціальних мереж (VK, OK) у робочий час (з 10:00 до 17:00).
3.2. Категорично заборонено використання торрент-трекерів. При виявленні трафіку Torrent обліковий запис блокується автоматично.
